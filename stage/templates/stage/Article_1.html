Machine Learning

Machine Learning is a sub-field of Artificial Intelligence. In general, a machine learning problem considers a set of n samples of data and then tries to predict properties of new unknown data. There are three main types of machine learning, plus hybrids of these three.

  1. Supervised Learning
  2. Unsupervised Learning
  3. Reinforcement Learning

Dataset

In Machine Learning, Dataset refers to the data which is given as input to an algorithm or use to test an algorithm's predictive accuracy.In other words, machine learning goal is to create computer softwares that learns from pass "experiences" and make predictions ( among other tasks ). The "experience" refers to the data that have been recorded by people and stored in a specific format. This data is what we refer to as a Dataset. Dataset comes in two parts.

  1. Features / Independent Variables
  2. Class / Dependent Variables
